# 计算机系统

操作系统（Operating System, OS）是计算机系统中最核心的系统级软件，它的核心职责是向下隔离底层硬件的复杂性，向上为应用提供统一、高效的资源管理与调度框架。

无论是 Windows、macOS 还是 Linux 这三大主流操作系统，其设计皆生根于经典的“冯·诺依曼架构”之上，所有的应用层软件都必须依附于它才能运行。如果将整个计算机知识生态比作一棵枝繁叶茂的大树，那么操作系统便是深扎于土壤中的底层根系。俗话说“万变不离其宗”，只要我们深入掌握了操作系统的底层原理，无论应用层的各种新框架、新语言如何眼花缭乱地更迭，我们都能做到“任凭风浪起，稳坐钓鱼台”，并能以此为支点，极快地触类旁通、迁移学习任何一项前沿技术。
