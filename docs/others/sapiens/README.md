相比于其他动物而言，**人类可以说是早产儿，许多重要器官发育还不够完善**。看看小马驹，出生没多久就能开始小跑；小猫出生不过几周，就能离开母亲亲子觅食；非洲大草原上的羚羊，可能刚出生就要躲避狮子的追逐。福祸相依，人类由于出生时发育尚不完全，相比其他动物，也**更能用后天的教育和社会化的方式加以塑造**。

人类在食物链的位置从中段跃居到顶端，导致了没有来得及发展天敌制衡。而狮子和鲨鱼花费了上百万年才站到食物链顶端，因此生态系统有时间发展出制衡措施来避免生态失衡。

智人相比于其他动物，在沟通方面，不仅仅是因为有语言，而是能够传达一些根本不存在的事物的信息，比如文字、金钱、国家、宗教、法律、公司等。这样的**虚构故事让智人得以即使素未谋面，只要共同相信某个故事，就能共同合作**（比如跨国公司的员工协作）。

采集社会相比农业社会，更能免受饥饿或营养不良的困扰，秘诀则在于多样化的饮食。此外，农业社会相对单一的食物来源容易因自然灾害引发饥荒，而采集社会则可以采集或狩猎其他食物，或迁移至受影响较小的地区。

在传染病方面，由于农业社会的人口密集、饲养的家禽家畜，更容易导致瘟疫流行，而采集社会的低密度和除了狗之外几乎不会饲养家禽家畜，降低了传染病的传播。

在人类拥有航海技术前，各种生物因为地理原因而相互隔绝，经过几百万年的磨合，他们达到了该地区内的生态平衡。而人类在学会了缝制衣服、制作武器、航海远行后，开始打破这种地理屏障，当登陆一个从未到达的地区时（如寒冷的西伯利亚、澳大利亚等与世隔绝的小岛），人类一跃成为顶级掠食者，从而对此处的生态造成破坏，导致大量生物的灭绝。

一个非常新颖的观点：不是人类驯化了小麦，而是小麦驯化了人类。小麦在被驯化前，不过是一种不起眼的野草，而由于它可以被驯化并为人类提供食物，而遍布全球各个大洲。同时人类之前经常迁徙去寻找食物，而种植小麦后则被固定在麦田旁边，他们需要为小麦浇水、除虫、拔草、赶鸟、施肥，全心全意为小麦服务才能获得回报。

人类驯化动物，对大多数动物来说则是悲惨的灾难，比如野生鸡的自然寿命是7~12年，而肉鸡则从孵化到流向肯德基只有几个月的时间，且一生都生活在狭窄的笼子里；奶牛则需要不停地怀孕，刚出生的小母牛被养大，小公牛则被宰杀；提供兽力的马、牛、驴等则为了抑止攻击性和提升劳动，则会将其阉割，并套上枷锁和用皮鞭来训练；

统治者通过征收农民的粮食，养活了政治、战争、艺术和哲学，建起了宫殿、堡垒、纪念碑和庙宇。

想象所建构出来的秩序总是有一夕崩溃的风险，因为这些秩序背后靠的都是虚构的故事，只要人们不再相信，一切就风云变色。**为了维持想象建构出来的秩序，必须持续投入大量心力，甚至还得掺入些暴力和胁迫的成分**。比如为了让民众不违反想象建构的秩序，国家就需要有军队、警察、法院和监狱不分昼夜地发挥作用。

阿拉伯数字是印度人发明的，只是阿拉伯人攻打印度时发现了这套系统，然后加以改良传到中东，进而传入欧洲。

**农业革命之后，人类维系社会秩序的虚构故事即为“文化”**。

实现“世界一家”的基础：**金钱、宗教、国家**

计划经济的特征：各尽所能的最小值， 各抢所需的最大值。

金钱的基本特征：**人人都想要**。正因为有了金钱概念，财富的转换、储存和运送都变得更容易也更便宜，后来才能发展出复杂的商业网络以及蓬勃的市场经济。要是没有钱，市场和商业网络的规模、活力和复杂程度都必然相当有限。

金钱制度的两大原则：**万物可换**、**万众相信**。

无论是贝壳还是美元，它们的价值都只存在于我们共同的想象之中，因为所有参与其中的人都达成了共识，因此金钱才得以发挥其标注的价值。**金钱正是有史以来最普遍也最有效的互信系统**。

铸造伪币的罪行一直比其他欺诈行为判得更重。因为造伪币不只是单纯的欺诈，更是对主权的挑战，直接冒犯了国王的权力、特权和他本人。

就算是在宗教上水火不容的基督徒和穆斯林，也可以在金钱制度上达成同样的信仰。原因就在于宗教信仰的重点是自己相信，但金钱信仰的重点是“别人相信”。

在所有人类创造的信念系统之中，只有金钱能够跨越几乎所有文化鸿沟，不会因为宗教、性别、种族、年龄或性取向差异而有所歧视。也多亏有了金钱制度，才让人就算互不相识、不清楚对方人品，也能携手合作。

**帝国是一种政治秩序**，有两项重要特征：

1. 帝国必须统治着许多不同的民族，不同的民族各自拥有不同的文化认同和独立的领土
2. 疆域可以灵活调整，而且可以几乎无限扩张

帝国并非需要专制皇帝（如大英帝国和美国就是民主政体）或规模很大（雅典帝国在巅峰时期面积和人口都不及今日的稀烂，但因统治了几十到数百个政体，而被认为是帝国）。

帝国正是造成民族多样性大幅减少的主因之一。**帝国就像一台压路机，将许多民族独特的多样性逐渐夯平**，整合制造出他们更大的新群体。

帝国政体其实非常稳定，多半时候要打倒反叛军根本不成问题。帝国之所以会倾覆，通常都是因为有外部侵略或是内部统治精英的内斗。相对而言，说到被征服者奋起追求自由、对抗帝国统治，向来记录都很少，他们多半都是持续臣服长达数百年之久。**通常这些民族会慢慢被帝国消化，最后自己独特的文化也烟消云散**。

很多时候，某个帝国崩溃了，并不代表属民就能独立。反而是每在帝国瓦解或遭到驱逐之后，就会由新的帝国取而代之继续统治。

现代西方认为公正合理的世界应该是由各个独立的民族国家组成，但古代中国自秦朝之后，则认为政治分裂不仅动荡不安，而且被看做是黑暗时代。因此每次一个帝国朝代崩溃，这种政治理论主流就让各方竞逐的势力不安于各自为政，而一心追求统一。而且事实证明，最后总能统一，只是时间早晚的问题。

同化的过程常常带着痛苦和创伤。要放弃熟悉且深爱的地方传统并不容易，而要了解及采用新的文化也同样困难而令人深感压力。雪上加霜的是，即使帝国的属民千辛万苦终于接受了帝国文化，也可能需要数十年甚至数百年，帝国的精英才能把他们看成是“我们”。

地球上现存的文化已经没有任何所谓纯净的文化，现存的所有人类文化，至少都有一部分是帝国和帝国文明的遗绪。**就算我们真的要完全去除掉某个残暴帝国的遗绪，希望能够重建并维护在那之前的“纯正”文化，很有可能最后恢复的也不过是更古老、更残暴的帝国留下的文化**。

自公元前200年左右，大多数人都已经活在各个帝国之中。看来，未来很可能所有人类就活在单一的帝国之下，而且这会是个真正的全球性帝国。**统一全球这件事，很可能已经离我们不远**。时间来到21世纪，民族主义正在迅速失去地位。越来越多人相信，**真正的政治权威应该是来自所有人类，而不是某个特定国籍的成员**，而人类政治的方向也该是保障人权，维护全人类的利益。如果确实如此，那么现在全球有将近200个独立国家，就反而形成阻碍。

宗教必须同时具备“普世特质”（任何人种、地域均可信仰，不会排外）和“推广特质”（将其信念尽可能传播给更多的大众）。直到公元前 1000 年前，才开始出现具备普世和推广特质的宗教。这可以说是史上最重要的革命之一，**对于人类的统一所做的重大贡献，绝不亚于帝国或金钱。**

很多古代神话其实就是一种法律契约，在这份契约中，人类承诺会永远崇敬某些神灵，以换取人类对其他动植物的控制权。

真正让科学革命起步的伟大发现，就是人类发现自己的无知。在宗教统治的时代，宗教是不容置疑的，穆罕默德宣称自己知道全部的真相，而信众称呼他为“先知的封印”，而达尔文不会说自己是“生物学家的封印”、自己完全解开声明的谜团。

**古希腊**和**古罗马**是两个历史上非常重要的文明古国，对后来数千年的欧洲文化和全球文化都产生了深远的影响，分别存在于以下的时期：

- 古希腊文明：大约存在于公元前8世纪到公元前6世纪初。这一时期被称为古典时期，包括了许多著名的城邦，如雅典、斯巴达、科林斯等。在这个时期，古希腊文化取得了很大的成就，如哲学、政治、科学、艺术等领域都有着重要的发展，并对后来的欧洲文化产生了深远的影响。
- 古罗马文明：大约存在于公元前753年到公元476年。这一时期涵盖了罗马共和国和罗马帝国两个阶段。在这个时期，罗马文化和政治逐渐壮大，征服了地中海地区和欧洲大陆的大部分领土，在建筑、工程、法律、军事等方面作出了很多重要的贡献，并在其后的欧洲历史上留下了不可磨灭的印记。

古罗马与罗马帝国时间轴

罗马帝国是世界历史上一个伟大的帝国，无论经济、文化、政治和军事上的成就都达到很高的水准，并和在与公元前一世纪兴起于亚洲的汉帝国西、东遥相并立。**罗马帝国幅原辽阔，而且国祚长久，使拉丁希腊的语言、文化、宗教、发明、建筑、哲学、法律及政府模式对后世的影响相当深远**（如今天的医学中大量的拉丁文可追溯至罗马帝国时期；许多欧洲语言也都以拉丁语作为其基础，如英语、法语、西班牙语）。

在西罗马帝国灭亡后，欧洲进入中世纪（476~1453，始于西罗马帝国灭亡、终于东罗马帝国灭亡）黑暗时代，欧洲陷入战争、饥荒、瘟疫的动荡，在这种状况持续了*一千年*后，终于迎来了**文艺复兴**：

1. **人文主义**的兴起：文艺复兴时期推崇人类本身，而不是宗教或政治权威。这种思想被称为人文主义，强调个人和人类尊严、自由和创造力。这种理念推动了西方现代社会和文化的发展。
2. **科学与技术的进步**：文艺复兴时期促进了科学和技术的进步。许多思想家开始研究自然现象并制定实证方法进行研究。这导致了一些重大发现，例如伽利略发现地球围绕太阳旋转的事实。
3. **艺术和建筑的繁荣**：文艺复兴时期是欧洲艺术和建筑的巅峰时期，体现了人文主义的理念。许多著名的艺术家和建筑师，如达芬奇、米开朗基罗和拉斐尔等，都在这个时期繁荣发展。
4. **社会和政治制度的改革**：文艺复兴时期也是社会和政治制度的变革时期。例如，马基雅维利在其著作《君主论》中提出了一种新的政治思想，强调权力的重要性。

文艺复兴（14世纪末到16世纪初）标志着欧洲走出了自西罗马帝国覆灭以来持续千年的动荡混乱，重新走向了繁荣。接下来的三次工业革命（1760~1840，机械革命；1870~1945，电气革命；1945~，信息革命）和大航海时代（15 世纪末到 16 世纪，葡萄牙、西班牙、荷兰、俄国、法国、英国依次崛起）奠定了欧洲今天的世界地位，并持续影响至今，如英语成为了今天使用最广泛的语言，巴西、澳门使用葡萄牙语，墨西哥、阿根廷等国家仍将西班牙语作为主要语言或官方语言；英法仍有多个海外属地；加拿大、澳大利亚、新西兰仍是英联邦的成员国（且澳、新国旗中仍有英国国旗标志）。

葡萄牙与西班牙分别是最早海外扩张的两个国家，双方签订《托德西利亚斯条约》划分海外势力范围：大西洋以东（含巴西）归葡萄牙，以西归西班牙。葡萄牙殖民帝国因里斯本大地震而走向衰落。之后英国、法国、荷兰相继崛起，最终英国击败法国西班牙联合舰队（特拉法尔加广场纪念该海战），一跃成为海上霸主，开始了其日不落帝国的征程。

虽然德国已经投降，但日本还在负隅顽抗。美国军队作势攻入日本本岛。日本誓死抵抗，准备决一死战，而且这绝非装腔作势。美国将军告诉杜鲁门总统，如果真要入侵日本本土，必然有超过百万美国士兵丧命，战争也必然会延长到1946年。于是，杜鲁门决定使用这款新型炸弹。在两枚原子弹投下之后，日本宣布无条件投降，战争就此告终。

科学不会假装自己知道未来“一定”会有什么。只有宗教和意识形态会声称自己知道这些答案。

中国和波斯其实并不缺乏制作蒸汽机的科技（当时要照抄或购买都完全不成问题），**他们缺少的是西方的价值观、故事、司法系统和社会政治结构**，这些在西方花了数个世纪才形成及成熟，就算想要照抄，也无法在一夕之间内化。之所以法国和美国能够很快跟上英国的脚步，是因为他们本来就和英国共享一套最重要的故事和社会结构。而**中国和波斯总是追赶不及，则是因为整个关于社会的想法和组织就是不同**。

究竟欧洲在现代早期培养了什么潜力，让它能在现代晚期称霸全球？这个问题有两个相辅相成的答案：**现代科学和资本主义**。

**过去的帝国主义者都认为自己已经了解了整个世界**，“征服世界”只是为了要利用及传播他们自己对于世界的看法。古罗马人、蒙古人和阿兹特克人之所以积极四方征讨，**为的是权力和财富，而不是为了新知**。相较之下，欧洲帝国主义之所以要前往遥远的彼岸，除了为了新领土，也是为了新知识。在18、19世纪，几乎每一趟从欧洲出发的军事远征队都必定有科学家同行，科学家的目的不在打仗，而是科学研究。例如拿破仑1798年进攻埃及的时候，就带了165位学者。

科学就像一把让人类通向拥有无穷力量之门的钥匙，谁掌握了这把钥匙，谁就能掌握自己的命运。

因为视野狭隘而付出沉重代价的，并不只有美洲原住民而已。在亚洲当时的各大帝国（奥斯曼土耳其、波斯帝国、莫卧儿帝国以及中国）很快就听说欧洲似乎有了重大发现。然而，他们对这件事却没什么兴趣，还是继续相信这个世界是以亚洲为中心旋转，完全没打算和欧洲人争夺美洲或大西洋、太平洋的新航道。

真正让银行（以及整个经济）得以存活甚至大发利市的，其实是我们对未来的信任。“信任”就是世上绝大多数金钱的唯一后盾。

但到了现代，贵族已经被新精英分子取代，这批新人都是资本主义教条的信徒。过去的公爵侯爷（贵族会把财富用于奢华消费，比如举办各种比赛和宴会、发动战争、投入慈善、兴建宫殿和教堂，而不是提升庄园的产量，或者寻找新的市场）黯然退位，取而代之的是董事、金融家、实业家。

比如对于狼群来说，如果它们觉得作为猎物的羊群会无限扩大，岂不是荒谬至极？然而，人类的经济在整个现代时期就是这样不可思议地持续指数增长。唯一的原因，就在于科学家总是能每隔几年就取得另一项发现，提出另一项发明，像美洲大陆、内燃机引擎，或者运用基因工程的羊。如果实验室的脚步不敌印钞泡沫破灭的速度，则会产生巨大的经济危机。

虽然知道 18 世纪晚期，全球的经济强权仍掌握在亚洲手中，但欧洲的信贷制度让资本不眠不休，高效运转，驱动着投资者不断扩张以寻求更高的回报，因此在 19 世纪后，欧洲成为了全球经济中心。

哥伦布和麦哲伦航行均由西班牙王室赞助，这为西班牙带来了丰厚的回报，让西班牙成为了欧洲最富有的国家之一，并与葡萄牙在海外殖民分庭抗礼。

由于荷兰对投资者遵守信用，并且司法独立，而西班牙虽然更为富有和强大，但却经常赖账，导致对西班牙的投资越来越少，最终荷兰取代了西班牙的地位。荷兰成立了世界上第一个股份制公司——荷兰东印度公司，而之后的英国同样通过英国东印度公司的方式快速扩张。

印度从 1498 年被葡萄牙建立第一个殖民地开始，之后荷兰、法国、英国依次殖民印度，直到 1947 年才独立，被殖民统治长达四个半世纪！由于长时间的殖民统治，印度内部矛盾重重，比如种姓制度原本是为了方便殖民者统治，即使独立七十多年，该制度仍荼毒印度。在印度独立后，又再次分裂：东边独立出孟加拉国、西边独立出巴基斯坦。

在工业革命前，能量转换只能是植物通过光合作用把太阳能转换为有机化合物，然后动物摄取植物中的能量转换为肌肉运动。因此人类历史一直由植物的生长周期和太阳能的变化周期（昼夜和四季）主导。而**工业革命的核心，其实就是能源转换的革命**。

工业革命影响人类社会的重大变化：**家庭和地方社群崩溃**（由国家和市场取代）、**父权社会的解体**、**时间工业（标准）化**、都市化、农民阶级消失、工业无产阶级兴起、民主化、青少年文化。

工业革命前，家族式的社会结构兼顾了福利制度、卫生体系、教育系统、建筑产业、劳工工会、退休基金、保险公司、广播、电视、报纸、银行，甚至警察功能。而工业革命后，这些功能都被市场化，从而瓦解了家族内的凝聚力。

|  | 个人力量 | 家庭和社群力量 | 国家和市场力量 |
| --- | --- | --- | --- |
| 前现代循环 | **↓** | **↑** | **↓** |
| 现代循环 | **↑** | **↓** | **↑** |

二战后战争减少的原因：

1. 核武威慑：两个拥核国家开战无异于集体自杀
2. 以前的财富是土地、资源、人口等，占领或掠夺都很方便，而国际化的今天，财富的形式变为了人力资本、科技知识，以及像银行这样的复杂的社会经济结构，想要占领或掠夺相当困难。
3. 全球国际化使得各国间都相互依赖，战争并不能让某方的利益大增，而长久的和平却能让两国通过贸易繁荣
4. 古代尚武的首领才能减少外敌的入侵，而国际化的今天则是以政治家、外交家等文人主导
