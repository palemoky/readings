# 📚 Readings - 藏经阁

> 个人知识库，记录技术底层逻辑与生活思考。

这里是我的数字花园和阅读沉淀。不只是摘抄原文，更多的是提炼核心概念、记录实践思考以及构建个人的知识体系。

## 本地启动预览

```bash
uvx zensical serve
```

## 开发环境设置

```bash
# 安装 WebP 转换工具
brew install webp

# 安装 pre-commit
brew install pre-commit

# 启用 Git hooks（仅需执行一次）
pre-commit install
```

> 配置完成后，commit 时会自动将 PNG/JPG/GIF 图片转换为 WebP 格式并更新 Markdown 引用。
